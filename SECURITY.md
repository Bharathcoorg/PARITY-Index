# Security Policy

## Supported Versions
Security fixes apply to the latest release on `main`.

## Reporting a Vulnerability
- Please email security disclosures to the maintainers (add contacts here).
- Provide reproduction steps, impact assessment, and affected components.
- We will acknowledge receipt within 72 hours and coordinate a fix timeline.

## Severity and Handling
- Critical/High: hotfix with an advisory; may pause deployments.
- Medium/Low: scheduled for the next minor release.

## Scope
Contracts, off-chain services, and deployment scripts. Do not submit secrets.
